<b><h1>Телеграм бот авто барахолки</h1></b>

В данный момент не работает - @avto73ru_bot

Суть:
Пользователь путем общения с ботом создаёт объявление, которое публикуется в паблик. Так же пользователь может настроить фильтры и получать уведомления в боте о конкретных объявлениях.
Бот выполняет роль модератора, типизируя объявления и валидируя от спама. Запоминает пользователей для рассылки по фильтрам.


    -Реализован на Laravel 12
    -Бизнес-логика вынесена из контроллера в Service(в будущем разнесу от туда по классам)
    -2 таблицы. 1 хранит информацию о пользователях 2 хранит информацию о составляемом объявлении.
        Связь Один к одному реализована в моделях
